minesweeper

Mines:

Rules

numbers indicate how many mines are in the surrounding 8 squares

left-click an unknown square to uncover it

right-click an unknown square to flag it as a mine

left-click a number square to:
a) uncover all surrounding squares (if the number in the square matches the number of surrounding flags)
b) flag all uncovered surrounding squares (if the number in the square matches the number of uncovered surrounding squares)
